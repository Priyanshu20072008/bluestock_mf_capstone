# Bluestock Mutual Fund Analytics

## Overview

This project analyzes Indian mutual fund data using Python, SQL, SQLite, and Power BI.

## Features

- Data Ingestion
- Data Cleaning
- Live NAV Fetching
- SQLite Star Schema
- SQL Analytics
- Exploratory Data Analysis
- Power BI Dashboard

## Tech Stack

- Python
- Pandas
- SQLAlchemy
- SQLite
- Plotly
- Seaborn
- Power BI
